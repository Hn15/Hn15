## Implementation Strategy

Use a large markdown heading with styling

### 2. Statistics Pie Chart

- HTML: 50% (orange/red)
- CSS: 30% (blue)  
- Java: 10% (orange)
- MySQL: 10% (blue)

### 3. Currently Using Section
- Use Arch Linux logo (can use an image or icon)
- Add "currently using" text

### 4. Scrolling Message + GIF
- Use animated text or a creative section
- Add a fun GIF (using Giphy or similar)
